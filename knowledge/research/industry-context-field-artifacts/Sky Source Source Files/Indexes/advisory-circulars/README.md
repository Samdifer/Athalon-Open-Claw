## Citation Format

```
AC {number}[-{suffix}], Section {section}[.{subsection}]
```

**Examples:**
- `AC 145-4A` - Full document reference
- `AC 145-4A, Section 5` - Specific section
- `AC 43-9D, Section 4.2.1` - Detailed reference

## Regulatory Status

Advisory Circulars are **non-regulatory**. They describe acceptable methods of compliance but:
- Are not the only acceptable methods
- Do not create additional requirements beyond CFR
- Provide best practice guidance
