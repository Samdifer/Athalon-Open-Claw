## Overview

The Data Collection Tool (DCT) forms are standardized evaluation instruments used by FAA Aviation Safety Inspectors to assess repair station compliance with 14 CFR Part 145 requirements.

### Form Naming Convention
```
EP_{category}_{subcategory}_{type}_145F_AW_V{version}.PDF

EP = Examination Program
145F = Part 145 Repair Station (Foreign designation possible)
AW = Airworthiness
V{n} = Version number
```
