## Data Collection Tool (DCT)

The DCT forms are the primary evaluation instruments. Located in:
`Source Files/SAS FS DCT Data collection tools/`

### DCT Form Naming Convention

```
EP_{category}_{subcategory}_{type}_145F_AW_V{version}.PDF
```

- `EP` = Examination Program
- `145F` = Part 145 Repair Station (Foreign certification may be 145)
- `AW` = Airworthiness
- `V{version}` = Version number

### Key DCT Categories for Part 145

| Form Prefix | Purpose |
|-------------|---------|
| EP_1_4_x | Initial certification inspections |
| EP_1_5_x | Secondary evaluations |
| EP_4_x | Continued operations surveillance |

## Citation Format

```
FAA Order 8900.1, Vol {volume}, Ch {chapter}, Sec {section}
```

**Examples:**
- `FAA Order 8900.1, Vol 6, Ch 9, Sec 4` - Training surveillance
- `FAA Order 8900.1, Vol 2, Ch 5` - Air agency certification

## Regulatory Status

FAA Order 8900.1 provides **internal FAA guidance** for inspectors. It:
- Interprets CFR requirements for field application
- Contains both mandatory and optional evaluation items
- Defines the DCT evaluation criteria
