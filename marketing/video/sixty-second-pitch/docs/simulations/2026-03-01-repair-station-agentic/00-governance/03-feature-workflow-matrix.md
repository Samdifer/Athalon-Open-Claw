# Feature Workflow Matrix

## Core Feature to Workflow Mapping

| Feature Group | Real Workflow Anchor | Primary Team | Supporting Teams |
| --- | --- | --- | --- |
| Dashboard / My Work | Shift kickoff and task pull | Alpha | Echo |
| Work Orders / Tasks | Induct -> execute -> sign -> release | Alpha | Charlie |
| Scheduling / Capacity | Backlog -> bay assignment -> rebalance | Bravo | Alpha, Echo |
| Parts Receiving / Inventory | Receive -> inspect -> stock or quarantine | Charlie | Alpha |
| Rotables / Loaners / Tool Crib | Component swap and tooling controls | Charlie | Alpha, Bravo |
| Billing Quotes / Labor Kits | Estimate -> draft quote -> convert | Delta | Bravo |
| Invoices / Payments / AR | Bill closeout and collect cash | Delta | Echo |
| Compliance AD/SB / Audit Trail | Fleet risk review and drill-down | Charlie | Echo |
| Reports / Forecast / Profitability | Station management decisions | Echo | Delta |
| Settings / Locations | Multi-station governance and segregation | Echo | Bravo |

## Cross-Feature Journeys Required

- Schedule-to-quote handoff
- Work-order-to-parts handoff
- Work-order-to-compliance gate
- Quote-conversion-to-invoice handoff
- Multi-location planning and resource partitioning
